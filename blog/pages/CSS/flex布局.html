# Flex布局

# flex基本概念

  • 容器具有这样的特点：父容器可以统一设置子容器的排列方式，子容器也可以单独设置自身的排列方式，如果两者同时设置，以子容器的设置为准。
    1. 轴——主轴，交叉轴
    2. 容器——父容器，子容器

# flex基本属性

# 一.父容器设置属性（以父容器作为布局基准）

  • 设置子容器沿主轴排列：justify-content——用于定义如何沿着主轴(横轴)方向排列子容器，包含以下属性值：
    1. flex-start：起始端对齐
    2. flex-end：末尾段对齐
    3. center：居中对齐
    4. space-around：子容器沿主轴均匀分布，位于首尾两端的子容器到父容器的距离是子容器间距的一半
    5. space-between：子容器沿主轴均匀分布，位于首尾两端的子容器与父容器相切
  • 设置子容器如何沿交叉轴排列：align-items——用于定义如何沿着交叉(纵轴)轴方向分配子容器的间距，包含以下属性值：
    1. flex-start：起始端对齐
    2. flex-end：末尾段对齐
    3. center：居中对齐
    4. baseline：基线对齐，这里的 baseline 默认是指首行文字，即 first baseline，所有子容器向基线对齐，交叉轴起点到元素基线距离最大的子容器将会与交叉轴起始端相切以确定基线
    5. stretch：子容器沿交叉轴方向的尺寸拉伸至与父容器一致

# 二.子容器设置属性

  • 在主轴上如何伸缩：flex属性，包含以下属性值：
    1. 无单位数字——比例
    2. 带单位px的数字
    3. none不伸缩
  • 单独设置子容器如何沿交叉轴排列：align-self，包含的属性同align-items保持一致。

# 三.在父容器上设置flex轴（主轴，交叉轴）

  • 主轴伸缩方向
    1. 从左向右：flex-direction: row
    2. 向左：flex-direction:row-reverse
    3. 从上向下：flex-direction: column
    4. 向上：flex-direction: column-reverse

# 四.在父容器上设置是否换行

  • 对应属性：flex-wrap，属性值如下：
    1. nowrap：不换行
    2. wrap：换行
    3. wrap-reverse：逆序换行,沿着交叉轴反方向排列
    4. 轴向与换行组合设置：flex-flow,相当于 flex-direction 与 flex-wrap 的组合
    5. 多行沿交叉轴对齐：align-content,当子容器多行排列时，设置行与行之间的对齐方式

# 五.子容器上设置是否伸缩

  • 设置基准大小：flex-basis——表示在不伸缩的情况下子容器的原始尺寸。主轴为横向时代表宽度，主轴为纵向时代表高度
  • 设置扩展比例：flex-grow——子容器弹性伸展的比例
  • 设置排列顺序：order——改变子容器的排列顺序，覆盖DOM结构中的顺序，默认值为 0，可以为负值，数值越小排列越靠前
