# 原生常用API总结

# URL类

  • eg-url = 'https://b.jianbing.com/act/home/huodong/20190318.php?p=jianbing#reload'
  1. window.location.href——url
  2. window.location.origin——https://b.jianbing.com(域源)
  3. window.location.host——b.jianbing.com(域名)
  4. window.location.protocol——https(协议)
  5. window.location.pathname——b.jianbing.com/act/home/huodong/20190318.php(URL名称)
  6. window.location.search——?p=jianbing(URL参数)
  7. window.location.hash——#reload(Hash参数)

# CSS选择器类

  • 这两种DOM选择符查找的是静态的DOM元素，取出之后不会随着文档流中元素的变化而变化
  1. document.querySelector(selectors)——selectors是对应的选择器字符串,查找出对应的元素组中的第一个
  2. document.querySelectorAll(selectors)——查找出对应的所有元素的节点集合(NodeList)
  • 以下选择器查找的是动态的DOM元素，每次都会重新查询一次文档流中DOM节点
  1. document.getElementById(id)——通过元素的id进行查找
  2. document.getElementsByTagName(标签名)——通过标签名查找对应的元素集合，是一个数组
  3. document.getElementsByClassName(类名)——通过类名查找对应元素集合
  4. document.styleSheets[0].addRules(ele, style)——向对应的元素添加特定样式，特别是伪类

# DOM操作类

  • Element.getBoundingClientRect()——返回元素的属性以及相对于视窗的位置

    1. 返回的是一个对象，对象包含[width, height, left, top, right, bottom]等属性
    2. 一般主流浏览器都支持

  • 获取视窗宽高

  // 获取视窗宽高的兼容写法
  let clientWidth = document.documentElement.clientWidth || window.innerWidth
  let clientHeight = document.documentElement.clientHeight || window.innerHeight

# 日期类兼容

  • new Date()设置日期在IOS中的兼容问题
  let time = new Date('2019-03-18 17:55:00') //这种‘-’日期连接方式在IOS中不支持
  // debug
  let time = new Date('2019-03-18 17:55:00'.replace(/-/g, '/'))
  // 即用'/'代替'-'对日期进行定义即可
