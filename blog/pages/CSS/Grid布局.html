# Grid布局总结——网格布局👪

# Grid布局-基本概念

  • 同Flex布局一样，掌握Grid布局需要了解它的一些基本概念

# 一.容器和项目

  • 容器是最外层采用网格布局的区域，项目是容器内部采用网格定位的子元素

容器——外层父元素

项目——内层子元素

# 二.行和列

  • 水平区域是行，垂直区域是列

# 三.单元格

  • 行和列交叉的区域就是单元格

# 四.网格线

  • 用于划分单元格的线就是网格线，在浏览器内审查元素的时候会看到对应的网格线,网格线的条数是对应的行列数+1

# 容器属性

  • 这些属性是针对外围的容器，直接作用于容器上面的

# 一.display属性

  • 用于指定一个容器为网格布局
div {
  display: grid;
  display: inline-grid; // 外围容器变为行内元素,内部元素还是网格布局
}

# 二.grid-template-columns和grid-template-rows属性

  • 用于设置每一列的列宽和每一行的行高
div {
  display: grid;
  grid-template-columns: 100px 100px 100px; // 列宽——属性值的个数表示一行划分的列数
  grid-template-rows: 100px 100px 100px; // 行高
  /* 即可以使用绝对单位，也可以使用百分比 */
  grid-template-columns: 33.33% 33.33% 33.33%;
  grid-template-rows: 33.33% 33.33% 33.33%;
  /* 简介写法 */
  grid-template-columns: repeat(3, 33.33%);
  grid-template-rows: repeat(3, 33.33%);
  /* repeat重复某种模式 */
  grid-template-columns: repeat(2, 100px 20px 80px); // 表示(100px 20px 80px 100px 20px 80px)
  /* auto-fill自动填充 */
  grid-template-columns: repeat(auto-fill, 100px); // 表示每列宽为100px，并且尽可能地自动填充
  /* fr关键字表示划分比例 */
  grid-template-columns: 1fr 1fr; // 表示两列等列宽
  grid-template-columns: 150px 1fr 2fr; // 第一列150px,剩余地第三列宽度是第二列地两倍
  /* minmax表示一个长度区间 */
  grid-template-columns: 1fr 1fr minmax(100px, 1fr); // 第三列列宽不小于100px,不大于前两列地宽度
  /* auto表示自适应宽度,用于双飞翼布局 */
  grid-template-columns: 100px auto 100px; // 左右定宽，中间自适应宽度
  /* 传统布局实例 */
  grid-template-columns: 70% 30%; // 左右分栏布局
  grid-template-columns: repeat(12, 1fr); // 传统地12分栏布局
}

# 三.grid-row-gap和grid-column-gap和grid-gap属性

  • 用于设置单元格之间地间距
.container {
  grid-row-gap: 20px; // 行间距
  grid-column-gap: 20px; // 列间距
  grid-gap: 20px 20px; // 行，列间距
}

# 四.grid-auto-flow属性

  • 用于设置容器子元素地排列顺序，默认排列顺序是先行后列，也可以设置为先列后行
.div {
  grid-auto-flow: row; // 先行后列
  grid-auto-flow: column; // 先列后行
  grid-auto-flow: row dense; // 先行后列,并尽可能地紧密填满，尽量不出现空格
  grid-auto-flow: column dense;
}

# 五.justify-items和align-items属性和place-content属性

  • 写法相同,包含以下属性
start end center stretch
对齐单元格的起始边缘 对齐单元格的结束边缘 单元格内部居中 拉伸，占满单元格的整个宽度（默认值）
  • place-content属性是两个属性地综合——align+justify

# 六.grid-auto-rows和grid-auto-columns属性

  • 用于指定多余行和列地宽高。
.container {
  display: grid;
  grid-template-columns: 100px 100px 100px;
  grid-template-rows: 100px 100px 100px;
  grid-auto-rows: 50px; // 多余一行地行高
}

# 项目属性

  • 针对子元素项目地属性

# 一.grid-column-start,grid-column-end,grid-row-start和grid-row-end属性

  • 使用了下面四个属性如果发生项目重叠后，可以用z-index设置项目的重叠顺序
  1. grid-column-start属性：左边框所在的垂直网格线
  2. grid-column-end属性：右边框所在的垂直网格线
  3. grid-row-start属性：上边框所在的水平网格线
  4. grid-row-end属性：下边框所在的水平网格线
.item-1 { // 可以用于设置单元格所占的位置大小,
  grid-column-start: 1;
  grid-column-end: 3;
  grid-row-start: 2;
  grid-row-end: 4;
  /* span关键字表示跨越网格线数量 */
  grid-column-start: span 2; // == grid-column-end: sapn 2;
}

# 二.grid-column和grid-row属性

  • 是上面四种属性的简写合并模式
.item-1 {
  grid-column: 1 / 3;
  grid-row: 1 / 2;
}
/* 等同于 */
.item-1 {
  grid-column-start: 1;
  grid-column-end: 3;
  grid-row-start: 1;
  grid-row-end: 2;
}
