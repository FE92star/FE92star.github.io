# 面试CSS布局总结

# 一.水平居中元素方案

  1. 行内元素——text-align:center;；
  2. 确定宽度的块级元素——margin:0 auto;或者绝对定位+负margin；
  3. 宽度未知的块级元素:
 a {
   display: table;
   margin: 0 auto;
 }
 a {
   display: inline-block;
   text-align: center;
 }
 a {
   position: absolute;
   left: 50%;
   transform: translateX(-50%);
 }
 a {
   display: flex;
   justify-content: center;
 }

# 二.垂直居中元素方案

  1. line-height——适用于纯的文字元素
  2. flex布局
  3. 绝对定位
  4. table布局

# 三.使用display: inline-block产生的影响

  • 两个行内块元素并排排列中间会有一个默认的间距，产生的原因是：元素之间的换行符或者间距被处理成空白符，占有一定的位置，解决的方式：
  1. 标签写在同一行，不回车；
  2. 父元素设置font-size:0;；
  3. 子元素用浮动布局。

# 四.圣杯布局的方案

  • 定义：左右定宽，中间自适应的布局方式为圣杯布局，常见的方式：
  1. flex布局，左右宽度一定，中间盒子设置为：flex: 1即可；
  2. float布局，三个盒子都浮动，再对左右的盒子用-margin进行处理；
  3. 绝对定位布局：左右的盒子分别为left:0;和right:0;，中间的盒子的left和right分别为左右两个盒子的宽度；
  4. grid布局。

# 五.设置一个盒子的高度永远是宽度的50%

div {
  width: 100%;
  height: 0;
  padding-bottom: 50%;
}
