# 《第十一章——Dom扩展》

# 选择符API

  • 根据CSS选择符匹配文档中的DOM元素，包含两种：
  1. querySelector接收一个参数为CSS选择符（标签，类名，id），一般是通过Element类型调用该方法，查找的是对应元素的后代元素，返回的是第一个匹配的后代元素,没有找到的话就直接返回null
  document.querySelector('body')
  document.body.querySelector('#canvas')
1
2
  1. querySelectorAll接收的参数同querySelector一样，不同的是它返回的是所有符合条件的NodeList的实例，是一个数组。
  document.body.querySelectorAll('div') // 查找body下面的所有div
  document.querySelectorAll('p strong') // 查找p下面的所有strong

2
3

# 与类相关的扩展

  1. getElementsByClassName()接收一个参数，包含一个或者多个类名的字符串，返回的是匹配的NodeList类型。
  document.getElementsByClassName('className haha')
  // 查找匹配'className'和'haha'的元素
1
2
  1. classList属性，dom元素的专有属性，表示类名集合，包含下面四种方法:
  2. add：添加类名；
  3. contains：是否包含指定参数的类名，如果包含返回true，否则返回false；
  4. remove：移除类名；
  5. toggle：类名集合中包含就删除并返回false，没有就添加并返回true。

# HTMLDocument扩展

  1. document.head
  2. document.charset
  3. H5规定可以为元素添加非标准的属性，在元素标签内添加data-，为元素添加语义信息，常常用于图片懒加载技术，将对应的图片的请求地址先放到自定义的data-src属性中，等到滚动到对应的图片位置时，将src属性的值填充即可加载对应的图片。

# 插入标记

  1. innerHTML用于读写调用元素的所有子节点的HTML标记。
  2. outerHTML用于读写该调用元素和所有子节点的HTML标记。
  3. scrollIntoView用于将对应的元素滚动到视窗之内，参数可以为空，或者为true/false，用法如下
  document.getElementById('id').scrollIntoView(true) // true底部平齐，false顶部平齐
