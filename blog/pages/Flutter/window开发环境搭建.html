# Flutter在window系统下开发地环境搭建

# Flutter的应用

  • 背景：Flutter是一种类似于RN的跨端项目，由Google基于Dart语言进行开发的一套跨端开发的解决方案
  • 发展：2017年刚开始出来，到现在两年多的时间，其在Github上面的仓库已经被star四万多颗，并且国内大厂的应用包括淘宝的闲鱼项目，目前还处于高速发展阶段，官方还在快速迭代版本。

# 对应环境搭建步骤

  1. 安装Flutter SDK-最高效的方式是直接从github仓库直接拉取当前最稳定的版本——git clone -b stable https://github.com/flutter/flutter.git
  2. 下载稳定的SDK之后，不能安装到需要很高权限才能访问的地址中，然后配置对应的环境变量，按照官网的配置，将官方为中国搭建的临时镜像配置到用户变量或者环境变量都可以，同时需要将Flutter的bin文件夹的安装安装地址配置到Path环境变量中去，然后就可以运行flutter doctor检测flutter的环境是否齐全。
export PUB_HOSTED_URL=https://pub.flutter-io.cn
export FLUTTER_STORAGE_BASE_URL=https://storage.flutter-io.cn

  1. 下载Android Studio作为开发的编辑器，地址为https://developer.android.google.cn/studio，为谷歌官方地址，下载安装之后会引导安装最新的Android SDK，这是flutter为安卓开发所必须的
  2. 安装SDK的时候，由于是外网的原因导致无法正常下载，所以按照https://blog.csdn.net/qq_23599965/article/details/80910202其中的第二种方案修改Host文件，ping查看国内哪个IP地址对接谷歌服务器最快，就将该地的IP地址加入到host文件中去即可；修改host文件的方式——https://blog.csdn.net/yuanlaijike/article/details/79668711，修改之后如何生效——https://blog.csdn.net/zl1zl2zl3/article/details/83614745
  3. 在编辑器plugins中下载Flutter和Dart两个插件
  4. 下载Android模拟器，主要用的是genymotion，这个是甲骨文公司开发的模拟器，顺带会安装虚拟机
  5. 安装JAVA环境，安装的版本是1.8.0版本的 jre jdk，下载地址——https://www.oracle.com/technetwork/java/javase/downloads/index.html，对应的是Java SE 8u211/212按照网上教程配置一堆的环境变量，安装地址不能更改，默认安装即可，否则有问题
  6. 根据提示快速创建flutter项目，同时将模拟器对接到编辑器中去
  7. run项目，打开模拟器即可运行项目，支持热更新
